          *** 04_비동기_Promise ***
    new Promise((resolve, reject) => {}).then((response)=>{}) // 비동기 작업 성공시(resolve)
                                        .catch((error)=>{})   // 비동기 작업 실패시(reject)
                                        
    1. 비동기 기본 함수
    function sub_process_executor(val) {
      return new Promise((resolve, reject) => {
        setTimeout(() => {
          resolve(val);
        }, 2000)
      })
    }

        *** Async / Await 를 이용한 비동기 처리로직 순서 배치 ***

    - async 와 await 를 사용해 비동기 처리에 순서를 부여할 수있다.
    - 22번을 먼저 출력하고 2000ms(2초)후 11번을 출력
    async function sub_process_order() {

      await sub_process_executor(22).then((response) => {
        console.log('sub2', response);
        sub2ThreadEl.innerHTML += `sub2...${response}`;
      }).catch((error) => { })

      await sub_process_executor(11).then((response) => {
        console.log('sub1', response);
        sub1ThreadEl.innerHTML += `sub1...${response}`;
      }).catch((error) => { })
    }
    sub_process_order();

    1. HTML 구조 (뼈대 잡기)
    Swiper가 움직이려면 특정한 클래스 네이밍이 아주 중요해요. 클래스명 변경xx!
    .swiper: 슬라이더의 전체 틀이에요. (CSS에서 크기를 잡는 곳)
    .swiper-wrapper: 슬라이드들을 한 줄로 길게 세워두는 기차 칸 같은 존재예요.
    .swiper-slide: 실제 내용(이미지)이 들어가는 각 칸이에요.
    .swiper-pagination & .swiper-button-prev/next: 페이지 번호와 화살표 버튼이 들어갈 자리예요.
    
    2. JavaScript 설정 (생명 불어넣기)
    - new Swiper(...) 부분을 통해 슬라이더에 지능을 넣어주셨어요.
    - const mainBannerSwiper = new Swiper('.swiper', {
      direction: 'horizontal',      //vertical:수직 , horizontal(수평)
      autoplay: {
      delay: 2000,
      pauseOnMouseEnter:true,
      },
      loop: true,
      effect: "slide",             // slide(기본), fade , scroll, cube , coverflow , flip
      mousewheel: true,
      
      pagination: {                // 페이지 넘기는 효과
        el: '.main-banner>.swiper>.swiper-pagination',
        clickable: true,
        type: 'fraction', //bullets(기본), fraction , progressbar
      },
      
      navigation: {                // 페이지 버튼 추가
        nextEl: '.main-banner>.swiper>.swiper-button-next',
        prevEl: '.main-banner>.swiper>.swiper-button-prev',
      },
    });
