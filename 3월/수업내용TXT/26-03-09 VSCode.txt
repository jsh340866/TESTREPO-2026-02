### Form
    <!-- 
        form : 사용자로부터 특정정보를 받아 서버로 전달하는데 사용되는 태그
        action Attribute : 전달받는 서버 URI
        method Attribute : 서버로 요청하는 방식
            - GET        : 사용자 요청 정보를 Query String으로 전달(Default)
            - POST       : 사용자 요청 정보를 Request body(Payload) 로 전달
            - PUT        :
            - PATCH      : 
            - DELETE     : 
    -->

    여러 Input 타입
	- text,password : 일반 문자열과 비밀번호 입력
	- radio : 여러 선택지 중 하나만 골라야 할 때 사용
	- checkbox : 중복선택 혹은 동의 여부에 사용
	- select & option : 드롭다운 목록에서 항목을 선택할 떄 사용

    Bootstrap 활용
	- 복잡한 설정 없이도 여러 스타일을 만들수 있다.
	- container / wrapper : 전체적인 콘텐츠의 폭을 잡고 중앙에 배치
	- row : 가로 줄을 생성
	- col, col-4, col-auto : 한 중 안에서 칸의 너비 조정 ( 전체 합은 12)
	- form-control, form-select : Bootstrap 전용 스타일을 입히는 디자인 생성

    사용자 편의성
	- label의 for와 input의 id 연결 : 글자만 클릭해도 체크박스나 입력창에 포커스가
	가도록 만들어서 사용자가 훨씬 편하게 느끼도록 함.
	- placeholder 대신 레이아웃 배치 : 레이블과 인풋을 나란히 배치해 가독성을 높힘.


### DataSet

    <div data-username="user1234" data-baban="babana">계정명</div>
    <div data-addr="대구시">주소</div>
    <div data-phone="010-1223-1232">연락처</div>
	- data-* (속성) : HTML 태그 안에 자바스크립트에서 꺼내 쓸 나만의 커스텀 데이터를 몰래 저장해요.
	- dataset : 태그에 숨겨둔 data-* 값들을 객체 형태로 한꺼번에 읽어오는 자바스크립트 도구예요.
 

### Etc-input

        <form action="/test.jsp">
        <div>
            <label for="">TEXT</label>
            <input type="text">
        </div>
        <div>
            <progress id="file" value="70" max="100"></progress>
        </div>
	- type="search": 검색어 입력 전용창으로, 글자를 한 번에 지울 수 있는 'X' 버튼을 제공해요.
	- type="url": 입력값이 웹 주소 형식(http:// 등)인지 브라우저가 자동으로 검사해 줘요.
	- type="number": 숫자만 입력 가능하며, 위아래로 값을 조절하는 스핀 버튼이 나타나요.
	- type="date": 클릭 시 날짜를 편하게 고를 수 있는 달력(Calendar) 팝업을 띄워줘요.
	- type="time": 시간을 선택할 수 있는 전용 시계 UI를 브라우저가 생성해 줘요.
	- type="datetime-local": 날짜와 시간을 하나의 입력창에서 동시에 선택할 수 있게 해줘요.

### Block형과 Inline형
      블록(Block) 요소 (예: <div>)
        - 특징: 기본적으로 가로 전체 영역을 차지해서, 다음 요소는 항상 새로운 줄에서 시작
        - 스타일 적용: width, height, margin, padding 등 모든 크기 관련 속성이 자유롭게 적용돼요.

        인라인(Inline) 요소 (예: <span>)
        - 특징: 필요한 만큼의 가로 공간만 차지하며, 다른 요소들과 같은 줄에 나란히 배치돼요.
	

### StylePriority
	- !important**라는 아주 강력한 속성, 최고 우선순위, 자주 사용 x
	- 인라인 스타일 (<h1 style="...">): 태그 안에 직접 적은 스타일입니다.
	- 내부 스타일 (<style> 태그): 문서 안에 작성된 스타일입니다.
	- 외부 스타일 (<link>): 별도의 .css 파일에 작성된 스타일입니다.

### Background
    <style>
        *{box-sizing: border-box;}        
        body{margin: 0; padding: 0;}
        .wrapper{height: 5000px;}
        .bg{
            position: absolute;
            top: 800px;
            width: 100%;
            height: 400px;
            background-image: url('./images/bg.jpg');
            background-repeat: no-repeat ;
            background-size: contain;
/* cover : 전체 채우기 , contain : 이미지 비율에 맞게 표시 */
            background-attachment: fixed;           
        }
    </style>
	- background-attachment: fixed 배경을 고정
	- cover와 contain의 차이를 아는 것이 중요


### TextBasic
    <!-- 글자간격 -->
	- letter-spacing : 글자 사이의 간격을 조절해요. (자간)
	- word-spacing : 단어 사이의 간격을 조절해요. 

    <!-- 글자위치 -->
	- line-height : 줄 높이를 조절해요. height 값과 동일하게 맞추면 세로 가운데 정렬
	- text-align : 텍스트를 가로 방향(left, center, right)으로 정렬해요.

    <!-- text-decoration -->
	- overline: 글자 위에 선을 그어요.
	- underline: 글자 아래에 선을 그어요
	- line-through: 글자 중간에 취소선을 그어요.

    <!-- font-weight -->
    	- font-weight : 100부터 900까지: 숫자가 클수록 글자가 더 굵어져요. 보통 400은 보통 굵기, 
	  700은 진하게(bold)로 인식됩니다.
