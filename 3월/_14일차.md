  1. 변수지정 : let, 상수지정 : const
    let name; // 선언만 함
    name = "승원"; // 나중에 초기화(값 대입) 가능!
    const birthday = "0316"
      - let ≒ 자바의 초기화와 같다, 자바의 일반 변수 (값 변경 가능)
      - const ≒ 자바의 final 변수 (한 번 초기화하면 변경 불가능)

  2. Object : 현실 세계의 사물을 코드로 표현한 것, **상태(속성)**와 **행동(기능)**을 하나로 묶어 관리
    const myCar = {
      // 속성 (Property) : 객체의 데이터를 담는 변수 (key: value)
      owner: "정승원",
      curSpeed: 0,
      
      // 기능 (Method) : 객체 내부에서 정의된 함수 (메서드)
      Accel: function() {
        this.curSpeed += 10;
        console.log("가속!", this.curSpeed);
      }
    };
  3. document. 조작 - html과 연결하기
    - querySelector('선택자'): CSS 선택자(.class, #id, tag)를 사용하여 요소를 딱 하나 집어옵니다.
    - getElementById('id'): 특정 ID를 가진 요소를 빠르게 가져옵니다.

  4. 배열함수 ArrayObject (중요!)
    - .push(): 배열의 맨 마지막에 새로운 요소를 추가합니다.
    - .pop(): 배열의 맨 마지막 요소를 제거하고 그 값을 반환합니다.
    - .forEach(()=>{})  모든 요소를 순회        반복문(for) 대신 깔끔하게 요소를 하나씩 꺼내 씁니다
    - .sort(()=>{})     요소 정렬              (a, b) => a - b (오름차순), (a, b) => b - a (내림차순)
    - .filter(()=>{})   특정 조건만 추출        조건에 맞는 요소들만 모아 새로운 배열을 만듭니다.
    - .reduce(()=>{})   요소를 하나로 응축      누적합, 누적곱 등 배열 전체를 계산해 단 하나의 값을 만듭니다.
    - .amp(()=>{}) : 한 놈씩 꺼내서 변신시킨 다음 새로운 리스트를 만드는 것.
