## 화면구현

Client : 요청하는 주체(요청자)	Server : 물리적인것 (컴퓨터)
		                          Serve : 제공하다
                              Server : (Service) 를 제공하는 주체

IP : 네트워크 연결을 위한 주소

포트 : 서비스 접근을 위한 번호 ex) 웹브라우저 = 80번 포트, (index.html)

HTML : Hyper Text Markup Language(마크업 문서를 만들기 위한 언어)

HTTP : Request(Server에 요청), Response(Client에게 응답)

웹 표준 치침 : 전자정부 웹사이트 이용자가 특정 운영 체제나 웹 브라우저에 상관없이 접속할 수 있도록, 정부에서 전자정부 시스템 구축 시 반영해야 하는 최소한의 규약을 정의한 것이다.

웹의 3요소 : 웹 표준, 접근성, 호환성

대표언어
 - HTML : 구조만들때
 - CSS : 스타일링 
 - Javascript(가장중요) : 동적 처리(브라우저의 기능사용) - 객체지향문법 JAVA

### 기본 용어정의 

	lang = 기본언어
	charset = UTF-8
	viewport = 본문 화면 창
        <!DOCTYPE html>     :   이 문서가 HTML5 문서임을 선언    
        <html lang="ko">    :   HTML문서의 시작을 나타내며  lang 속성을 이용하여 문서의 기본언어가 한국어임을 명시
        <head>              :   문서의 메타데이터와 제목등을 포함하는 머리말
        <meta charset="UTF-8"> : 문서의 문자 인코딩을 UTF-8 으로 설정
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        : 반응형 웹 디자인을 위한 뷰포트 설정
        : width=device-width : 뷰포트의 너비를 디바이스의 너비만큼 설정   
        : initial-scale=1.0 : 페이지가 처음 로드될때 기본 확대/축소 수준을 지정 
        <title>Document</title>                    
        <body>              :   문서의 본문, 브라우저에 ViewPort(웹페이지를 사용자가 보는영역)에 표시되는 내용

### 태그 설명
    <hr/> : 구분선을 만듬


    <h1 style="border:1px solid">HELLO WORLD</h1>
    <h2>HELLO WORLD</h2>
    <h1></h1> 
	- 한 행 전체를 사용하는 블럭라인
	- 타이틀 제목으로 주로 사용함 
	- h1,2,3,4 여러개를 추가로 만들수 있고 소제목이 됨


    <p></p>
	- 본문의 내용의 문단을 나눌때 사용, 단락구분용


    <div style="background-color: orange;width:300px;height: 300px;border: 5px solid;padding:20px;margin:20px;"></div>
    <div></div>
	- 블랙형 기본태그
	- 여러 스타일을 추가해 사용할 수 있음.
	- 본문 내용에 박스를 만들수 있음.


    <span>HELLO WORLD</span><br/>
    <span>HELLO WORLD</span>
    <span style="display:inline;background-color: orange;width:300px;height: 300px;border: 5px solid;padding:20px;margin:20px;"></span>
    <span></span>
	- 한 행을 공유하는 인라인
	- <br/> 태그를 사용해 줄을 바꿀수있음
	- 여러 스타일을 부분적으로 사용할 수 있음.
	- 블럭에서는 가능하지만 인라인에서는 불가능한 것이 존재


    <!-- ul-li : 순번없는 리스트 ul>li*4{List_$}-->
    <ul>
        <li>List_1</li>
        <li>List_2</li>
        <li>List_3</li>
        <li>List_4</li>
    </ul>
    <ul></ul>
	- 순번없는 리스트를 만들수 있음
	- ul>li*4{List_$} 를 사용해 여러 리스트를 빠르게 만드는 코드도 존재


    <!-- ol-li : 순번있는 리스트 : type=a,i,1.. //ol(order list)-->
    <ol type="I">
        <li>List01</li>
        <li>List02</li>
        <li>List03</li>
        <li>List04</li>
    </ol>
    <ol></ol>
	- 순번있는 리스트를 만들수 있음
	- 5가지의 순번이 있음( I,  i, A, a, 1 )
	- <ol type="A">를 사용해 순번을 부여할 수 있음.
	

    <li></li>
	-리스트를 만드는 태크
	-네이게이터를 만들때 가장 많이 사용하게될 태그
