

```
Client : 요청하는 주체(요청자)	Server : 물리적인것 (컴퓨터)
		                          Serve : 제공하다
                              Server : (Service) 를 제공하는 주체

IP : 네트워크 연결을 위한 주소

포트 : 서비스 접근을 위한 번호 ex) 웹브라우저 = 80번 포트, (index.html)

HTML : Hyper Text Markup Language(마크업 문서를 만들기 위한 언어)

HTTP : Request(Server에 요청), Response(Client에게 응답)

웹 표준 치침 : 전자정부 웹사이트 이용자가 특정 운영 체제나 웹 브라우저에 상관없이 접속할 수 있도록, 정부에서 전자정부 시스템 구축 시 반영해야 하는 최소한의 규약을 정의한 것이다.

웹의 3요소 : 웹 표준, 접근성, 호환성

대표언어
 - HTML : 구조만들때
 - CSS : 스타일링 
 - Javascript(가장중요) : 동적 처리(브라우저의 기능사용) - 객체지향문법 JAVA
```
