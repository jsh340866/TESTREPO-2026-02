### Table 차트,표
    사용되는 태그
	- table : "이제부터 표를 그릴 거야!"라고 선언하는 전체 울타리
	- th : Table Header의 약자로, 각 칸의 **'제목'**을 넣을 때 사용 (글씨가 굵고 가운데 정렬)
	- td : Table Data의 약자로, 실제 **'내용(데이터)'**을 넣는 일반적인 칸
	- tr : Table Row의 약자로, 표의 **'행(가로줄)'**을 만들 때 사용 (표는 항상 줄 단위로 작성)
	- rowspan="숫자" : Row Span의 약자예요. **아래(세로)**로 칸을 합칠 때 사용
	- colspan="숫자" : Column Span의 약자 **옆(가로)**으로 칸을 합칠 때 사용
	- caption : 
		- 반드시 <table> 태그 바로 다음에, 즉 표의 첫 번째 자식으로 딱 한 번만 사용
		- 브라우저에서 기본적으로 표의 중앙 상단에 표시
	- thead : Table Header, 표의 **맨 윗줄(제목 열)**들을 하나로 묶어준다
	- tbody : Table Body, 표의 **실제 데이터(본문)**가 들어가는 부분
	- tfoot : Table Footer, 표의 **맨 아랫줄(요약, 합계 등)**을 묶어줘요.

    예시
    <table>
        <caption>SEMENTIC</caption>
        <!--어디에 위치하든 head는위 foot은 아래  -->
        <tfoot>
            <tr>
                <th rowspan="2">뿌리차</th>
                <th colspan="2">과일차</th>
                <td>6</td>
            </tr>
        </tfoot>
        <tbody>
            <tr>
                <td>1</td>
                <td>2</td>
                <td>3</td>
            </tr>
        </tbody>
        <thead>
            <tr>
                <th>1열</th>
                <th>2열</th>
                <th>3열</th>
            </tr>
        </thead>
    </table>


### ATag
    <!-- <a href="./07_Table.html">문서이동-1</a> -->
    <a href="javascript:void(0)">FUNCTION1</a>
    <a href="javascript:void(0)">FUNCTION2</a>
    	- <a> Anchor(닻) 약자로 문서 이동을 위한 버튼을 만들 때 사용한다.
	- href: Hypertext Reference, 어디로 갈지 목저지 주소를 적는 곳
	- href 값을 비워두면 기본적으로 페이지 상단으로 이동이다
	- 값을 비워둘때는 javascript:void(0) 부여해 아무것도 없음을 나타내야한다.
	

    <!-- target="_blank" -->
    <a href="https://www.google.com" target="_blank">구글을 새 창에서 열기</a>
	- 링크를 눌렀을 때 현재 창이 바뀌지 않고, 새 탭이 뜨게 하고 싶을 때 사용한다.


    <!-- title="" -->
    <a href="info.html" title="상세 정보 페이지로 이동합니다">정보 보기</a>
	-마우스를 링크 위에 올렸을 때 작은 설명창이 뜨게 해준다.

### Video
        1) video options
            controls : 하단제어부 표시 여부
            autoplay : 자동재생
            muted : 음소거
            loop : 반복재생
        2) MIME TYPE
        Client -Server 간 자료(보통 파일)을 교환할때 사용되는 표준화된 식별자
        type/subtype

        subtype : html, png, mp4, pdf ...


    예시 
    <div class="wrapper">

        <div class="gradient-overlay"></div>
        <div class="bg-video">
            <video controls autoplay muted loop>
                <source src="./video/Test.mp4" type="video/mp4" />
            </video>
        </div>
    </div>


### Form (중요)


    주요사용 태그
	- <div class="wrapper"> : 전체 내용을 하나로 묶어주는 커다란 상자
	- <h2>회원가입</h2> : 어떤 페이지인지 알려주는 큰 제목
	- <form action=""></form> : 사용자가 쓴 모든 정보(이메일, 비번 등)를 서버로 보내기 위한 큰 봉투
	- <div></div> : 
	- <label>기본정보</label> : "이 칸은 이메일 적는 칸이에요~"라고 알려주는 이름표
	- <span>필수 사항</span> : '필수 사항' 같은 작은 안내 문구를 넣을 때 써요
	- <input type="text" placeholder="닉네임" /> : type="text" / 일반적인 글자를 쓰는 칸
	- <input type="password" placeholder="비밀번호" /> :  type="password" placeholder /  
      입력창 안에 미리 적혀있는 회색 안내 글자
	- <input type="file" /> : type="file"/ 파일을 올리고 싶다면 사용하는 태그
	- <select></select> : 클릭하면 아래로 선택지가 툭 떨어지는 목록 박스 / option태그랑 같이 사용
	- <option value="">010</option> : 그 안에 들어있는 하나하나의 선택지
	- <button></button> : 모든 정보를 다 채운 후 꾹 누르는 마지막 버튼
