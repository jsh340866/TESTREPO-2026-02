    HELLO WORLD<br/>
    HELLO           WORLD<br/>

    <!-- < , > &lt;&gt; -->
    HTML 태그 기호 설명 &lt;span&gt;태그는 inline형 태그입니다.
	- <,> 태그로 사용되는 부호를 나타내기 위해 &lt; , &gt; 사용한다

	- &gt; = Greater Than( > , ~보다 크다) )


### Emmet 
    <!-- 자식 요소 선택 : '>' -->
    <!-- div>ul>li -->
    <div>
        <ul>
            <li></li>
        </ul>
    </div>
    태그>태그>태그>
	- 자식 요소를 선택해 빠르게 기본 구조를 만들수 있다.


    <!-- 형제노드 : '+' -->
    <!-- div>ol+ul -->
    <div>
        <ol></ol>
        <ul></ul>
    </div>
    태그+태그
	- 형제 노드를 추가해 같은 라인에 만들 수 있다.


    <!-- 상위태그 이동 : '^' -->
    <!-- div>ul>li^p -->
    <div>
        <ul>
            <li></li>
        </ul>
        <p></p>
    </div>
    태그^태그
	- 하위태그에서 상위태그로 빠져나가 이어갈수 있다.


    <!-- 태그의 내용추가 : '{}'  --> 
    <!-- div>ul>li*4>a{$.MAINMENU}+ul>li*5>a{$.SUBMENU} -->
    태그>태그^태그>태그{$.}
	- 간편하게 기본 구조를 만들수있다
	- 태그{} 중괄호 안에 내용을 넣을수있다
	- $ 를 사용해 추가되는 태그에 숫자를 부여할 수 있다.


    <!-- 클래스 선택자 : '.' -->
    <!-- .c1 -->
    <div class="c1"></div>
    <div class="abc"></div>
	- .
	- class="" 
	- 태그를 사용해 특정 그룹을 만들 수 있다.
	- 같은 클래스 이름을 여러 태그에 동시 줄 수 있다.


    <!-- ID선택자 : # -->
    <div id="wrapper"></div>
    <div id="id2"></div>
	- #
	- id=""
	- 클래스보다 높은 우선순위
	- 중복이 불가능하고 딱 한 번만 쓸 수 있다.

    <!-- 속성 : [] -->
    <!-- ol[type='I']>li*4{TEST_$} -->
    <ol type="I">
        <li>TEST_1</li>
        <li>TEST_2</li>
        <li>TEST_3</li>
        <li>TEST_4</li>
    </ol>
	- []
	- 대괄호 안에 특정 속성을 가진 태그만 골라서 가져올수 있다.


### Table 차트,표
    사용되는 태그
	- table : "이제부터 표를 그릴 거야!"라고 선언하는 전체 울타리
	- th : Table Header의 약자로, 각 칸의 **'제목'**을 넣을 때 사용 (글씨가 굵고 가운데 정렬)
	- td : Table Data의 약자로, 실제 **'내용(데이터)'**을 넣는 일반적인 칸
	- tr : Table Row의 약자로, 표의 **'행(가로줄)'**을 만들 때 사용 (표는 항상 줄 단위로 작성)
	- rowspan="숫자" : Row Span의 약자예요. **아래(세로)**로 칸을 합칠 때 사용
	- colspan="숫자" : Column Span의 약자 **옆(가로)**으로 칸을 합칠 때 사용
	- caption : 
		- 반드시 <table> 태그 바로 다음에, 즉 표의 첫 번째 자식으로 딱 한 번만 사용
		- 브라우저에서 기본적으로 표의 중앙 상단에 표시
	- thead : Table Header, 표의 **맨 윗줄(제목 열)**들을 하나로 묶어준다
	- tbody : Table Body, 표의 **실제 데이터(본문)**가 들어가는 부분
	- tfoot : Table Footer, 표의 **맨 아랫줄(요약, 합계 등)**을 묶어줘요.

    예시
    <table>
        <caption>SEMENTIC</caption>
        <!--어디에 위치하든 head는위 foot은 아래  -->
        <tfoot>
            <tr>
                <th rowspan="2">뿌리차</th>
                <th colspan="2">과일차</th>
                <td>6</td>
            </tr>
        </tfoot>
        <tbody>
            <tr>
                <td>1</td>
                <td>2</td>
                <td>3</td>
            </tr>
        </tbody>
        <thead>
            <tr>
                <th>1열</th>
                <th>2열</th>
                <th>3열</th>
            </tr>
        </thead>
    </table>

	- 값을 비워둘때는 javascript:void(0) 부여해 아무것도 없음을 나타내야한다.
